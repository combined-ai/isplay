## How To Read These Pages

Use concept pages for durable mental models. Use [guides](/docs/guides) when you need a task recipe. Use [reference](/docs/reference) when you need exact command names, route paths, enum values, fields, and limitations.

<Callout type="info" title="Evidence discipline">
Most isplay concepts exist to prevent overclaiming. A replay is not automatically proof, a fixture is not automatically reality, and an experiment with one trial is not automatically a statistically supported conclusion.
</Callout>
