## Records

| Schema | Purpose |
| --- | --- |
| `AnalysisRun` | Persisted summary and validity labels. |
| `EvidenceNode` | Prompt clause, retrieval, memory, state, tool fixture/schema, model config, token signal, intervention, or metric delta evidence. |
| `EvidenceEdge` | Relation between evidence nodes. |
| `EffectCandidate` | Ranked effect with status, score, labels, confidence, evidence refs, and next actions. |
| `ExperimentStatistics` | Counts, rates, confidence intervals, and metadata. |
