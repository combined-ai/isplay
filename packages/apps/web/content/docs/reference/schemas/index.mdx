## Shared Record Fields

Most durable records include:

| Field | Type | Notes |
| --- | --- | --- |
| `id` | string | Prefix-based IDs such as `project_`, `run_`, `event_`, `replay_`. |
| `createdAt` | string | ISO timestamp. |
| `metadata` | object | JSON metadata on most domain records. |
