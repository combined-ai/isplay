## Records

| Schema | Important fields |
| --- | --- |
| `Project` | `id`, `createdAt`, `name`, `metadata` |
| `Agent` | `projectId`, `serviceName`, `framework`, `packageVersions`, `metadata` |
| `Run` | `projectId`, `agentId`, `name`, `status`, `startedAt`, `endedAt`, `metadata` |
| `EventRecord` | `projectId`, `runId`, `seq`, `type`, `refId`, `parentEventId`, `occurredAt`, `data`, `metadata` |
| `Artifact` | `projectId`, `runId`, `kind`, `objectKey`, `sha256`, `sizeBytes`, `mimeType`, `compression`, `redactionState`, `metadata` |
